# Cpp-Project-Template
C++ General Project Template

## Layout
- `3rdparty`: Third-party and vendored dependencies that are consumed by the build.
- `include`: Public headers exposed by the repository.
- `src`: Application and library source files.
- `test`: Automated tests covering the project.
- `script`: Automation, build, and deployment helpers.
- `doc`: Design notes, user guides, and other documentation.
- `sample`: Sample apps or playgrounds for exercising components.
- `poc`: Proofs of concept that explore ideas not yet integrated in the main sources.
